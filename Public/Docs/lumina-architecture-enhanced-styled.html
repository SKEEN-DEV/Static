I. Introduction & Context

UI Introduction

ğŸ¯ Design & Implementation

I designed the intial phase of the Lumina project as a strategic overlay system - a specialized intelligence interface that operates on top of existing MCD infrastructure without disrupting core functionality. This approach introduces advanced AI-powered news, expert suggestions and chat intelligence while fully preserving the original MCD capabilities. Later on we can expand Lumina capabilites to be offered inside different UI modules to help users complete their daily tasks using components that follow the Lumina AI look and feel.

Implementation Strategy: Lumina functions as a cross-app system. Its entry point is placed in the Navigation app, while all operational logic resides in the MCD app. This keeps Lumina code organized and easy to maintain.

Visual Approach

To help the interface stand out from MCD, we utilised a new color palette and fonts based on the Pulsar Group brand style guide. The Pulsar Group designer provided us with a logomark and typeface to represent Lumina. Using the Pulsar Group look and feel helps it appear platform agnostic.

AI identifiers in the UI refer to how AI-powered features are distinguished from standard ones, helping users easily recognize when they are interacting with AI.

Key Design Considerations:

Implementation Strategy:

  • Visual Cues: Distinct styling, colors, and iconography for AI-powered components

  • Progressive Disclosure: AI features enhance rather than overwhelm the core interface
